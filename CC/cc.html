Flocking Simulation

The Coding Challenge is a challenge created by The Coding Train. Daniel Shiffman codes a different project every time, in either p5.js, or Processing. One of out projects this year was choosing a coding challenge,  completing it, and then teaching our class about it.

I learned a lot from this challenge.I had trouble understanding how the particles work, so there was a lot of tinkering happening. I refactored the original project, into a liquid-like simulation. Although it still follows the flocking rules. I learned about forces, particles, collisions, and gravity.
